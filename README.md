# 📚 BookStore

A full-stack bookstore application built with **Spring Boot, Spring Security, JWT, MySQL, React, and Cloudinary**.

## 🚀 Features

* User Registration & Login
* JWT Authentication & Authorization
* Book Management
* Cloudinary Image Upload
* Shopping Cart Functionality
* Responsive React UI

## 🛠 Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* JWT
* MySQL
* JPA/Hibernate
* Cloudinary

### Frontend

* React
* Vite
* Axios
* React Router
* Context API

## 📂 Project Structure
BookStore
├── BackEnd
└── FrontEnd

## ▶️ Run
### Backend
cd BackEnd
mvn spring-boot:run

### Frontend
cd FrontEnd
npm install
npm run dev
